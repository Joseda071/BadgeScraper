# BadgeScrapper


Find rare users in discord servers

## How to use
- Replace the guild_id, server_id and token by the values you wanna use
- If you never used discum before run this command in terminal : ```python -m pip install --user --upgrade git+https://github.com/Merubokkusu/Discord-S.C.U.M.git#egg=discum```
- then run the script normally: ```py badges.py```
